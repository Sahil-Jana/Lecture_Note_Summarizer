# AI Agent Prototype – Lecture Notes Summarizer

### 1. Overview
This project automates the task of summarizing lecture notes using a fine-tuned LLM with LoRA.

### 2. Files
- `Lecture_notes_summarizer.py` – main agent code
- `AI Agent Architecture.pdf` – architecture & workflow
- `AI Agent Data Science Report.pdf` – fine-tuning & evaluation details
- `Dataset.txt` – link for dataset - GNR649
- `interaction_logs.txt` – test conversations
- `README.md` – project summary and usage guide

### 3. How to Run
2. Run `Lecture_notes_summarizer.py` in Google Colab.
2. Use function `agentic_lecture_summarizer("your query")` to test.

### 4. Fine-Tuned Model
- Method: LoRA on a base T5 model
- Target: Improving summarization for academic notes
